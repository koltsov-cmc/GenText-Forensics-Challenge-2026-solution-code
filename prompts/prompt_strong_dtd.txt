==================================================================
TYPE CLASSIFICATION (apply by reading content inside the region)
==================================================================

For each DTD region, look at what the region contains in the OCR output:

* **Visual Clumsy** — region contains a visible editing artefact: blur,
  pixelation, white-out / black-out box, overpainting, JPEG halo,
  rectangular redaction.

* **Logical Fraud** — region contains a specific value (number, date,
  percentage, total, year, ID, name, amount). The value renders cleanly
  but is likely altered relative to what it should logically be.

* **Semantic Subtle** — region contains a clean word or phrase whose
  meaning contradicts the surrounding sentence (e.g. "for-profit
  charity", "permanent temporary contract", a wrong product name in a
  technical list).

If you cannot decide between Logical Fraud and Semantic Subtle:
  * Region contains DIGITS → Logical Fraud
  * Region contains LETTERS only → Semantic Subtle

If the region is empty/blank or obviously redacted → Visual Clumsy.
